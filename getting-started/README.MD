# Getting Started

My recommendation is to not limit yourself to one AI or tool. 

 The companies are working very hard to one up each other so its important to revisit the tools every so often. Maybe one month you will use claude code and the next use codex. Getting a feel for each tool will be beneficial in the long run. If someone asks me what my favorite tool or model is, I would say it depends. It's all about tradeoffs, do we want a cheaper model, a US centric model, a model that is better at Design? The truth is that its better to combine all the tools and use each model and tool at its full potential without getting to fixated on the tool. 
 
We are all in the same team so lets use all the tools in our disposal and see how it feels for us.
